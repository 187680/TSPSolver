# TSPSolver
Neural Network approaches for the Traveling Salesman Problem.
- Hopfield Network (Future)
- Elastic Nets
- Self-organizing map (WIP)

# Enviroment
| |version|
|:--|:--:|
|python|3.6.3|
|pip|9.0.1|
|matplotlib|2.1.0|
|numpy|1.13.3|

In the project directory, run the following command:  
```
pip install -r requirements.txt
```
